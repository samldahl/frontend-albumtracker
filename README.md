# Album Tracker

A React app for tracking your music albums and songs.
![Album Tracker Screenshot](./public/screenshot.png)

## Setup

1. Clone the repo and install dependencies:
   ```bash
   npm install
   ```

2. Create a `.env` file in the root with your backend URL:
   ```
   VITE_BACK_END_SERVER_URL=http://localhost:3001
   ```

3. Start the dev server:
   ```bash
   npm run dev
   ```

## Features

- Sign up / Sign in
- Create, edit, and delete albums
- Add, edit, and delete songs within an album
- View all albums and tracklists

## Tech Stack

- React 19
- React Router v7
- Vite
